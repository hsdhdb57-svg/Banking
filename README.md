# Мини-банкинг на Java + SQL

Небольшое консольное приложение, которое показывает работу банковских операций через SQL-базу.

## Что есть

- счета клиентов;
- HTML-сайт с главной страницей, личным кабинетом и просмотром базы;
- пополнение и снятие денег;
- переводы между счетами;
- банковские карты;
- журнал транзакций;
- история изменений;
- SQL-схема с внешними ключами и ограничениями;
- перевод выполняется внутри одной JDBC-транзакции.

## Технологии

- Java 17;
- Maven;
- JDBC;
- H2 Database, файловая SQL-база.

## Запуск

Запуск сайта:

```bash
mvn compile exec:java
```

После запуска откройте:

```text
http://localhost:8080
```

Если в Windows-консоли русские буквы отображаются неправильно, перед запуском можно включить UTF-8:

```powershell
chcp 65001
mvn compile exec:java
```

После первого запуска база появится в папке `data/`.

Консольную версию можно запустить так:

```bash
mvn compile exec:java -Dexec.mainClass=com.example.minibank.BankApp
```

## Структура базы

- `accounts` - счета и текущий баланс;
- `bank_cards` - карты, привязанные к счетам;
- `transfers` - отдельные записи переводов;
- `transactions` - денежные операции по счетам;
- `change_history` - журнал изменений сущностей.

SQL-схема лежит в `src/main/resources/schema.sql`.

## Основные файлы

- `WebBankApp.java` - HTML-сайт и обработка форм;
- `BankApp.java` - консольное меню;
- `BankService.java` - бизнес-логика и JDBC-запросы;
- `Database.java` - подключение к базе и создание таблиц;
- `schema.sql` - SQL-таблицы, индексы и ограничения.
