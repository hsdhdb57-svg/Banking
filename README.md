# Banking

Мини-банкинг на Java + SQL с простым HTML-сайтом.

## Структура

```text
Banking/
├─ pom.xml
├─ README.md
├─ .gitignore
├─ scr/
│  ├─ script/       Java-файлы
│  └─ bazadannih/   SQL-схема базы данных
├─ html/            HTML-страницы
└─ css/             стили сайта
```

## Запуск

```bash
mvn clean compile exec:java
```

После запуска открыть:

```text
http://127.0.0.1:8080/
```

## Что есть

- главная страница;
- личный кабинет по логину и паролю;
- счета, карты, переводы и транзакции;
- переводы по номеру карты получателя;
- страница базы данных с таблицами;
- SQL-база H2.

## Вход

Для новых счетов логин и пароль задаются при создании.

Для старых счетов после обновления автоматически создаются данные:

```text
логин: user<ID счета>
пароль: 1234
```

Например, для счета `1`: логин `user1`, пароль `1234`.
